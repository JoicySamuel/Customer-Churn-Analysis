# Customer-Churn-Analysis
A Power BI report that analyses the data of a broadband company 
This is a customer churn analysis for an internet service provider. This analysis is based on churn on factors like dependents, gender, tenure, payment method, and contract.

Some of the steps taken in the project are as follows:
1. Created a conditional column to get the churn in number format.
2. Replaced text values with numerical values in certain columns and changed their data type accordingly.
3. Find the sum of total services
4. Adding DAX formulas to create different measures.
5. Visuals used are cards, clustered column charts, tree charts, and pie and donut charts.
